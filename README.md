## How to use:

### Use the '-e' or '-encode' flags to encode a message

####    Sample input: `python3 main.py -e "hello world"`

####    Output: MOXJLQGBPL

### Use the '-d' or '-decode' flags to decode a message

####    Sample input: `python3 main.py -d wahoophlhislil`

####    Output: NICEANDDECODED
